TERRAINMAP
By: Alexander Guerrero
Date: 22/11/2015
*********************************************************************

This is my terrain map for made for Assignment 2 in my Computer
Graphics course. The algorithm randomly selects points and generates 
hills of random radius and height.

Make file included.

CONTROLS:

	w - toggle wireframe or solid view mode

	e - toggle flat or gouraud (smooth) shading

	l - toggle lights

		-> y, g, h, j - move lights position

	arrow keys - rotate map

	-/_ - zoom out

	+/= - zoom in

	r - reset map

	q - quit
